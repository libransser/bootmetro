# BootMetro


BootMetro provides simple and flexible HTML, CSS, and Javascript for web apps that wants to use the "Windows 8 MetroUI" style, without the need to run on Windows 8.
It is built on top of Twitter Bootstrap and HTML5 Boilerplate.


## Differences on this fork


For the moment, the differences between this particular fork and the original project in [https://github.com/aozora/bootmetro](https://github.com/aozora/bootmetro) are the change in the grid size from 12 to 24 columns, and modifications to include the responsive styles in one file.

So far the only changes were made in the /less folder. Examples have not been updated.
